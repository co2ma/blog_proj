# 📘 Personal Tech Blog (나만의 기술 블로그)

> **"배운 것을 기록하는 공간"**
>
> CS 지식, 알고리즘 풀이, 그리고 개발 과정에서의 트러블 슈팅을 정리하기 위해 직접 설계하고 개발한 개인 블로그 프로젝트입니다.

## 🎯 프로젝트 목적 및 배경

기존의 블로그 플랫폼을 대신하여, 다음과 같은 이유로 직접 블로그를 구축하게 되었습니다.

1.  **Full-Stack 역량 함양:** 아직 프론트엔드, 백엔드 모두 익숙하지 않은 기술이라 이 두 가지를 동시에 학습하면서 프론트엔드와 백엔드 로직을 모두 직접 구현해보며 웹 개발의 전반적인 데이터 흐름을 이해하고자 했습니다.
2.  **나만의 지식 아카이브:** 컴퓨터 공학(CS) 전공 지식, 알고리즘 문제 풀이 등을 체계적으로 정리하여 언제든 다시 꺼내 볼 수 있는 저장소가 필요했습니다.
3.  **기능 구현 경험:** 마크다운 렌더링, 태그 필터링 등 개발자에게 필요한 기능을 직접 기획하고 구현하는 경험을 쌓고 싶었습니다.

이외에도 여러가지 글을 작성 할 수 있는 나만의 폼이 있으면 좋겠다는 생각에 만들게 되었습니다.

## 🛠️ 기술 스택 (Tech Stack)

| 분류         | 기술 (Technology) |
| :----------- | :---------------- |
| **Frontend** | `React / Next.js` |
| **Backend**  | `Spring Boot`     |
| **Database** | `PostgreSQL`      |

## 📌 주요 기능 (Key Features)

-   **게시글 관리 (CRUD):**
    -   Markdown 문법을 지원하는 에디터 및 뷰어 구현
    -   게시글 작성, 수정, 삭제 및 조회 기능
-   **카테고리 및 검색:**
    -   CS, 알고리즘, 회고 등 카테고리별 분류 기능
    -   키워드 검색 및 태그 필터링
-   **UI/UX:**
    -   반응형 웹 디자인 (Mobile/Desktop)
    -   다크 모드 지원
